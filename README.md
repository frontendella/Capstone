# [Photo store](https://frontendella.github.io/Photo-e-commerce-website/)
A mock e-commerce website - user is able to choose some of the images to order for printing.

## Description

* useState()
* useEffect()
* useRef()
* useContext
* Routes, Route, Link
* PropTypes
* Custom Hook (useHover)
